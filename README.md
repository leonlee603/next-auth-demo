## Introduction

Auth.js is one of the most commonly used authentication libraries in Next.js projects. This is a small project integrating various tools to build a website login system.

## Stacks

- Framework: Next.js
- Authentication: Auth.js
- Database: Neon & Drizzle ORM
- Layout: Shadcn & Tailwind CSS
- Validation: Zod

## Project is hosted on Vercel

You can check the demo [here](https://next-auth-demo-six-nu.vercel.app/)
